# mario-mini

ブラウザで動く、**New Super Mario Bros風**テイストの 2D マリオ風ミニゲームです。

## 遊び方

1. `index.html` をブラウザで開く（またはローカルサーバーで配信）
2. 操作:
   - `←` / `→`: 移動
   - `Space`: ジャンプ
   - `R`: リスタート

## 特徴

- 赤帽子・青オーバーオール・ヒゲのマリオ風キャラクター描画
- パララックス背景（空・雲・丘）
- コイン収集、敵踏み、ゴール旗までの1ステージ

## ローカル実行例

```bash
python3 -m http.server 8000
```

開いたら `http://localhost:8000` にアクセスしてください。
