# mario-mini

ブラウザで動く、**New Super Mario Bros風**テイストの 2D マリオ風ミニゲームです。

## 遊び方

1. `index.html` をブラウザで開く（またはローカルサーバーで配信）
2. 操作:
   - `←` / `→`: 移動
   - `Space`: ジャンプ
   - `X`: ボスのしっぽをつかむ（離して投げる）
   - `R`: リスタート

## 追加した要素

- `?`ブロックを下から叩くとアイテム（キノコ）やコインが出現
- ブロックヒット時の飛び出しコイン演出とスコアのフローティング表示
- キノコ取得で `SUPER` 状態になり、一定時間パワーアップ
- 使用済み `?` ブロックの見た目変化、ブロックバウンス演出
- 雲ハザードが上空から降ってくる（接触でダメージ）
- ボス戦追加：しっぽをつかんで振り回し、投げ飛ばすとクリア

## ローカル実行例

```bash
python3 -m http.server 8000
```

開いたら `http://localhost:8000` にアクセスしてください。
